# Vocabulary Explorer Prototype

## Overview
This project is a high-fidelity interactive prototype for **Vocabulary Explorer**, a mobile app concept designed for Cree language learners, teachers, and expert users. The prototype supports searching for words, browsing themed vocabulary collections, saving custom collections, and switching between Basic and Expert modes for different levels of detail.

## Features
- Search for Cree words and themed vocabulary sets
- Browse topic-based collections such as Animals, Nature, and Body Parts
- View word details and related collections
- Save and manage custom collections
- Toggle between Basic and Expert modes
